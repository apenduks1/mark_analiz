Опис проєкту

Це веб-додаток для підрахунку статистики оцінок учнів у школі. Користувач може додавати оцінки, після чого система автоматично підраховує статистику: загальну кількість оцінок, середній бал та інші показники.

Проєкт створений для практики роботи з React та компонентною структурою.

Основні можливості

Додавання оцінок (наприклад: добре, нормально, погано)

Автоматичний підрахунок статистики

Показ загальної кількості оцінок

Підрахунок середнього значення

Відображення повідомлення, якщо оцінок ще немає

Технології

React

JavaScript

CSS

Component-based architecture

Структура компонентів

App – основний компонент, зберігає стан оцінок

FeedbackOptions – кнопки для додавання оцінок

Statistics – відображає статистику

Section – секція для відображення контенту

Notification – повідомлення про відсутність оцінок

Project Description

This project is a web application for calculating school students' grade statistics. Users can add grades, and the system automatically calculates statistics such as the total number of grades and the average score.

The project was created to practice React and component-based architecture.

Features

Add grades (for example: good, neutral, bad)

Automatic statistics calculation

Display the total number of grades

Calculate the average score

Show a notification if there are no grades yet

Technologies

React

JavaScript

CSS

Component-based architecture

Components Structure

App – main component that stores the state

FeedbackOptions – buttons for adding grades

Statistics – displays statistics

Section – content section

Notification – message when there are no grades
